# Skin Analysis App

## Run the App

**Terminal 1 - Start the server:**
```bash
python main.py
```

**Terminal 2 - Start the mobile app:**
```bash
cd uofthacks26
npx expo start
```

**On your phone:**
1. Open Expo Go app
2. Scan the QR code
3. Enter your computer's IP in the app (run `ipconfig` to find it)
